Overview

It is important to note that this project emerged from the UK based Startup company named Voxreel I cofounded together with Oliver Graham, William Graham and Mark Lloyd. Voxreels are reels of short and insightful video clips or “Voxes” expressing user experiences at a business.

The aim of this project was to develop a video sharing, mobile application, where users can record and share real experiences at various businesses with other users, culminating in a user generated film about a business. Most notably, the application allowed users to discover businesses and record short videos about different aspects of the venue, which are then compiled into searchable and insightful showreels. With this approach, we firstly hope to overcome the challenges in the landscape of current reviewing based applications by moving away from the concept of often corrupt, misleading and reputation damaging, written reviews and ratings. Secondly, we anticipated transforming the conventional method of discovering venues into a quicker and more engaging experience, by providing users with rich, informative and transparent video insights, consequently facilitating users’ decision making.

Features

  • Users can record and share short videos, annotating them with useful metadata.
  • Users can search for businesses using various searching algorithms.
  • Users can stream and browse through venues’ user generated video reels.
  • Users can search video reels for specific content.
  • Users can up-vote useful videos..
  • Users can report on not useful or damaging videos.
  • Users can create new businesses in case they have not yet been added to Voxreel.

Demonstration

Home

After installing and running the Voxreel app, an animating launching screen will appear while the configuration process takes place. Once completed, users will be presented with the home screen, i.e. the business search page. The user’s current position is highlighted as a blue wow circle on the map assuming he/she has allowed the application to track his/her location. For clarity, the items in the navigation bar illustrate the following pages (left to right): (1) business search page, (2) add video page, and (3) profile page.

Business Search

A search input view appears whenever a user specifies new details for a search. If no search has been triggered yet, the user is initially asked to specify a location. Otherwise, the application will reuse the location form the previous business search. Once a business search was triggered, its results are illustrated on either a map or as part of a list view, depending on which option is currently enabled.

Search input highlights

  • All autocompletion suggestions are performed via a fuzzy search algorithm, thus making up for misspellings.
  • Users can always edit the location by selecting the location label, consequently navigating back to the previous step.
  • Default locations (current location, current search map) and categories (bar, cafe, restaurant) are provided throughout each search input step in order to enable quick searching.
  • If a user decides to select a business item from the autocompletion list instead, the application will navigate the user straight to its business page (not shown in the video).

Map view highlights

  • To browse through businesses, a user can click between business map markers or swipe left and right within the red box.
  • Hand gestures can be used to navigate around and zoom in and out of the map view in order to gain a better understanding of the businesses’ locations.
  • Once a user begins navigating around the map view, a button appears above the red box, allowing the user to search for more businesses inside the current map view.

List view highlights

  • A user can scroll up and down the list to obtain a quick overview of all the businesses that were found. At its bottom, there is button that allows the user to create a new venue assuming he/she could not find the business he/she was looking for.

Map & List view highlights

  • A user can click the button right next to the search bar if he/she wants to switch between map and list view.
  • To further explore a business, a user can select the triangle inside a business item. The application will then navigate the user to the corresponding business page.
  • By tabbing the search bar, a user can modify his/her current search specifications and perform another business search.

Nearby Businesses

The add video page lists the nearby businesses around a user. Those are the businesses the user is currently allowed to record a video for, thus preventing him/her from uploading random video content to any venues.

Highlights

  • To capture a video for a business, a user can select any of the cross buttons. The application’s video recorder will then appear.
  • A user can tab the search bar and enter the name of the business he/she is looking for. Again, a fuzzy search string algorithm is applied to make up for misspellings.
  • If a user is not able to find a nearby business, the user can select the plus button inside the red box to quickly create the missing venue.

Seeding Businesses

To add a new venue and thereby contribute to the Voxreel business database, a user has to specify the name and location of the business as well as between one and three categories. Once submitted, those details will be verified and extended by the Voxreel team.

Capturing videos

The video capturing process comprises of recording a video, followed by annotating and finally sharing it with other Voxreel app users.

Recording highlights

  • To record a new video a user has to hold down his/her thumb inside the video square. A red progress bar will appear above the white menu, illustrating its currently recorded length (maximum 10 seconds). The recorder will notify the user in case the recorded video ended up being too short. After a successful recording, the video will be subsequently processed and cut down to a square format. If the user has not yet specified a theme for the video, he/she will be presented with the theme selector page to do so (this is illustrated in the video). Once all this is done, the video will begin playing in a loop.
  • Users can perform the following operations on a recorded video using the white menu box: delete the video by clicking on the trash icon, redo the video by selecting the reply icon or change the video theme by tabbing the edit icon.
  • To improve the usability of the app, a user can also record up to 4 videos in one session.

Annotation highlights

  • Users can specify a short message for each video, which will then be visible when other users watch the video as part of the business showreel.
  • Up to 5 hashtags can be specified to make the video more visible when searching the business for specific criteria.
  • Users can annotate the video with a price label depending on the selected video theme. For instance, an inside theme does not support a price label whereas a food theme does.

Upload highlights

  • Once a user triggers an upload for the newly captured video(s), the application will navigate the user back to the add record page and indicate the uploading process via a purple box popping up from underneath the navigation menu. Users are not required to wait for the data transfer to finish as the application will upload videos in the background. Once the upload has succeeded, the purple box will disappear. In case of a failure, a report message is being displayed inside the purple box with a button that allows users to reinitiate the upload.
  • If the application is being closed before uploads complete then they will resume the next time the application boots up again.

Business exploration

A traditional venue page consists of two sections, the showreel page and a page listing a business’s general information.

Highlights

  • Users can watch the business showreel by selecting the “Voxreel” tab or view any business related information by selecting the “Info” tab.
  • Users can swipe left and right to browse through the showreel. The active video immediately plays in a loop while listing its metadata and annotated information underneath.
  • Users can like/unlike each video by tabbing the heart icon.
  • Users can report each video by selecting the warning icon. A new page will appear, allowing them to compose and upload a short message to the Voxreel team.
  • Users can filter and order the business showreel by selecting the white box icon above the currently playing video. The application will present them with a settings page where they can select a number of sorting or filtering options they would like to apply to the business showreel.
