Overview

This was a research based project with the motivation to develop a Game With A Purpose (GWAP), named ‘Word Sheriff’, that would allow us to generate large collections of high quality annotations as part of an enjoyable gameplay by hiding away the annotation task from users completely. With this game we aimed to overcome shortcomings in previous GWAP design in order to improve game enjoyability and henceforth produce higher amounts of data in the long run. Based on our resulting data sets, we then anticipated in evaluating existing NLP methods but to also prove that they are reliable enough for endeavours in other fields in Computer Science.

The full project report including the assessment of our research results can be found here.

While Word Sheriff was its own student research project, its generated linguistic annotations fed into a more advanced NLP project lead by UCL researchers Pontus Stenetorp, Sebastian Riedel and Guillaume Bouchard, in which a novel way to evaluate AIs that communicate using natural language was introduced. More importantly, our resulting research paper was presented at the RepEval 2016 workshop for which we won a Facebook AI Research award.

Contribution

  • Developed the game frontend using traditional HTML, CSS and JavaScript on top of the Bootstrap framework.
  • Designed several game assets such as the game logo, menu boxes, background images, guesser and narrator related icons, etc.
  • Contributed to writing client side code (JavaScript) that connected server triggered events to the associated elements of the user interface. This comprised the following features: lobby menu at the start of each game, narrator and guesser timers, score updates as the game progresses, victory board at the end of each game to announce the winner and the remaining player rankings, player notifications when players guesses correctly or incorrectly, sound effects including the sound control menu, player icon updates based on their current role in the game.
  • Assisted in designing the database schema that allowed us to store game session data as well as player and annotation related information.
